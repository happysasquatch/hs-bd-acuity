Integrates Acuity Scheduling with WordPress for class management.

== Description ==

This plugin integrates Acuity Scheduling with WordPress, allowing you to display and manage your Acuity classes on your WordPress site.

Features include:

* Automatic synchronization of Acuity classes
* Custom post types for classes
* Customizable display templates
* Shortcodes for displaying classes
* Admin interface for managing classes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/acuity-integration` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Acuity API credentials in the plugin settings
4. Use the shortcode `[acuity_classes]` to display classes on your site

== Frequently Asked Questions ==

= How often are classes synchronized? =

By default, classes are synchronized hourly, but you can customize this in the plugin settings.

= Can I customize the display of classes? =

Yes, you can create custom templates in your theme to override the default templates.
